Instrucciones para construir un documento HTML

📄
  1. Crear un documento de texto con extensión html.
  2. Comenzar el documento con una etiqueta abierta de html.
    <html>
  3. Agregar la etiqueta head.
    <head></head>
  4. Agregar la etiqueta body, aquí se debe poner el contenido de la página.
    <body></body>
  5. Cerrar la etiqueta html.
    </html>
  6. Guardar el documento y abrirlo con algún navegador de Internet.
